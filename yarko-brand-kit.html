ЯРК групп
ВСЕ НЕОБХОДИМОЕ

Набор фирменных элементов, восстановленных из бренд-бука как чистый HTML + SVG. Каждый блок можно раскрыть и скопировать готовый код.

01 / ЗНАК

Фирменный знак

Зелёный бублик с центрированным крестом, который делит его на четыре части. Крест — прозрачные разрывы постоянной ширины, поэтому знак ложится на любой фон — на тёмном разрывы показывают фон, а не белый цвет. Градиентная версия — для крупных размеров, плоская — для мелких и favicon.

02 / ЛОГОТИП

Логотип и варианты

Локап собран как текст + инлайн-SVG (буква «О» = знак), а не картинкой — масштабируется размером шрифта, легко перекрашивается. Шрифт логотипа — Montserrat Light с широким трекингом (близко к оригиналу; при необходимости можно перевести в кривые для 100%-совпадения).

03 / ЦВЕТА

Фирменные цвета

Три базовых цвета из бренд-бука плюс два служебных оттенка зелёного для состояний и градиента. Нажмите на HEX, чтобы скопировать. Все цвета уже заданы CSS-переменными в начале файла.

04 / ШРИФТ

Типографика — Montserrat

Фирменный шрифт из бренд-бука — тонкий геометрический гротеск. Ближайший бесплатный аналог с полной кириллицей — Montserrat. Тонкие начертания (200–300) для заголовков, 400 для текста, 500 + капс + трекинг для подписей.

05 / ИКОНКИ

Категории продукции

Линейные иконки в едином стиле (viewBox 24, обводка currentColor, зелёный акцент — класс .acc). Цвет наследуется от родителя, поэтому одну иконку можно красить под тёмную/светлую тему.

06 / ИКОНКИ

Ценности и преимущества

Иконки для блоков «почему мы» и преимуществ. Тот же стиль и сетка, что и у категорий — для визуальной цельности сайта.

07 / UI

Кнопки и навигация

Базовые интерактивные элементы в фирменных цветах. Главная CTA «Стать партнёром» — зелёная заливка; вторичные — контурные.

08 / ДЕКОР

Декоративные элементы

Фоновые акценты с фирменного бланка: полупрозрачное кольцо-в-углу и точечная сетка. Подходят для хедеров, секций и футера.

ЯРКО ГРУПП · элементы фирменного стиля для веба. Шрифт — Montserrat (Google Fonts). Все ассеты — векторные, перекрашиваются через CSS-переменные.
